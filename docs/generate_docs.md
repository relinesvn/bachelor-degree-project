# Інструкція з генерації документації (Doxygen)

Для автоматичної генерації HTML-документації з нашого коду C++ використовується інструмент **Doxygen**.

### 1. Встановлення
* **Windows:** Завантажте інсталятор з [офіційного сайту Doxygen](https://www.doxygen.nl/download.html) і встановіть його. Переконайтеся, що `doxygen.exe` додано до системного `PATH`.
* **Linux (Ubuntu/Debian):** Виконайте команду `sudo apt-get install doxygen`.
* **macOS:** Виконайте команду `brew install doxygen`.

### 2. Локальна генерація документації
1. Відкрийте термінал у кореневій папці проєкту (там, де знаходиться файл `Doxyfile`).
2. Виконайте команду:
   ```bash
   doxygen Doxyfile
   ```
3. Після успішного виконання у папці проєкту з'явиться тека `html`.
4. Відкрийте файл `html/index.html` у будь-якому веб-браузері, щоб переглянути згенеровану документацію.

### 3. Перевірка якості документації (Linter)
У файлі `Doxyfile` активовано налаштування `WARN_AS_ERROR = YES`. Це означає, що Doxygen працює як суворий лінтер: якщо ви забули описати якийсь параметр (`@param`) або функцію, генерація документації завершиться з помилкою. Це гарантує 100% покриття коду документацією.
