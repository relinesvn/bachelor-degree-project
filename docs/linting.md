# Статичний аналіз коду (Linting)

## Обраний лінтер: clang-tidy
Для C++ обрано `clang-tidy`, оскільки він підтримується інструментарієм LLVM, інтегрований у MS Visual Studio та має найширший набір перевірок для сучасного стандарту C++11/14/20.

## Базові правила
Конфігурація зберігається у файлі `.clang-tidy`. Активовані групи правил:
* `readability-*`: покращення читабельності коду (іменування, магічні числа).
* `modernize-*`: рекомендації щодо використання сучасних конструкцій C++.
* `performance-*`: оптимізація швидкодії (наприклад, передача масивів за константним посиланням).
* `bugprone-*`: виявлення патернів, що часто призводять до помилок (наприклад, втрата точності).

## Запуск
Запуск здійснюється через вбудований аналізатор коду у Microsoft Visual Studio (Build -> Run Code Analysis) або через командний рядок:
`clang-tidy BachelorProject.cpp -config=""`
